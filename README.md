## Features

- Runtime language switching (no app restart needed)
- JSON-based translation files for easy maintenance
- Scalable pattern for adding new languages
- Demonstrates Flutter's localization best practices

## Tech Stack

Flutter, Dart, JSON translations

## Running

```bash
git clone https://github.com/thisisyoussef/localization_task.git
cd localization_task && flutter pub get && flutter run
```
